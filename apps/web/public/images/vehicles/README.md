# Vehicle Images

Place vehicle type images in this directory.

## Required Images

- `sedan.png` - Sedan vehicle image
- `suv.png` - SUV vehicle image  
- `truck.png` - Truck vehicle image
- `coupe.png` - Coupe vehicle image
- `hatchback.png` - Hatchback vehicle image
- `wagon.png` - Wagon vehicle image
- `van.png` - Van vehicle image
- `convertible.png` - Convertible vehicle image

## Image Specifications

- **Format**: PNG with transparency recommended
- **Dimensions**: Approximately 300-400px wide
- **Aspect Ratio**: Maintain consistent aspect ratio across all images
- **Background**: Transparent or white background
- **Orientation**: Side/profile view recommended for consistency

## Usage

These images are referenced in `src/lib/vehicles/vehicle-types.ts` and displayed by the VehicleTypeSelector component.
